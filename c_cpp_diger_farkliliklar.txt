C dilinde goto deyimi ile programın akışı bir bir değişkene ilk değer veren bildirimi atlayabilir. C++ dilinde bu durumu geçersizdir:
Aşağıdaki kod C'de geçerli ama C++ dilinde geçersizdir:

void func(void)
{
	goto nec;
	int i = 0;
nec:
	;
}
